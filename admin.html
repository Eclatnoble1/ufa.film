Bem-vindo ao painel da UFA FILM. Este sistema está conectado diretamente ao site principal.

1. Portfólio: Adicione novos vídeos usando o link de incorporação (Embed) do Instagram.
2. Imagens: Atualize as imagens fixas do site (Hero, Sobre, Serviços) colando novas URLs.

Nota: Como este é um site estático (GitHub Pages), utilizamos o armazenamento local do seu navegador para simular um banco de dados. As alterações ficarão salvas no seu computador.
